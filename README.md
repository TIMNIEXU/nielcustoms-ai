# NielCustoms.ai AI Customs Operating System

Full starter platform for:
- OCR Upload
- AI Tariff Calculator
- Customs AI Chatbot
- Importer Portal
- Broker Dashboard
- Automatic HTS Classification
- AI Compliance Risk Engine
- ABI / ACE integration placeholders
- Multi-user SaaS architecture
- API products

## Install
```bash
npm install
cp .env.example .env.local
npm run dev
```

## Vercel Environment Variables
- `OPENAI_API_KEY`
- `BLOB_READ_WRITE_TOKEN`
- `NEXT_PUBLIC_BASE_URL`

## Important Compliance Note
This is a technology starter. HTS classification, AD/CVD, PGA, Section 301, Chapter 99, ABI transmission, and ACE data use require licensed customs broker review and approved government/vendor access.
